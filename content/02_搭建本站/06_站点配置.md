# 站点配置

本站的所有可配置项集中在 `src/config/` 目录下，修改后保存即可热更新生效。

## 配置文件一览

| 文件 | 用途 |
|------|------|
| `src/config/site.ts` | 站点标题、导航栏、图标、主题、页脚、首页文案 |
| `src/config/profile.ts` | 关于页的个人信息：头像、简介、技能、项目、经历 |
| `src/config/navigate.ts` | 导航页的书签分类与站点链接 |

## 站点基础信息（site.ts）

```typescript
export const siteConfig: SiteConfig = {
  title: "Refinex Blog",
  subtitle: "老李的技术随想录",
  logo: {
    type: "image",
    imageSrc: "/logo.svg",
    href: "/",
  },
  // ...
};
```

### 导航栏

`nav` 数组定义顶部导航项，每项包含 `label`（显示文字）和 `href`（跳转路径）：

```typescript
nav: [
  { label: "首页", href: "/" },
  { label: "文档", href: "/docs" },
  { label: "导航", href: "/navigate" },
  { label: "工具", href: "/tools" },
  { label: "关于", href: "/about" },
],
```

### 主题设置

```typescript
theme: {
  enableToggle: true,       // 是否显示主题切换按钮
  defaultMode: "system",    // 默认主题：light / dark / system
},
```

主题状态持久化在 `localStorage` 的 `refinex-theme` 键中。

### 首页文案

```typescript
home: {
  title: "Refinex Blog",
  subtitle: "沉淀架构、后端与工程实践的系统化笔记。",
  searchPlaceholder: "搜索文章标题、描述或关键词…",
  primaryAction: { label: "Get Started", href: "/docs" },
  secondaryAction: { label: "导航入口", href: "/navigate" },
},
```

### 页脚

```typescript
footer: {
  copyright: "© 2026 Refinex Blog. All rights reserved.",
  meta: ["后端技术 · 架构思考 · 体验设计"],
  links: [
    { label: "隐私政策", href: "/" },
    { label: "使用条款", href: "/" },
    { label: "联系我", href: "/" },
  ],
},
```

### 图标

`icons` 数组配置顶部导航栏右侧的外链图标（如 GitHub）：

```typescript
icons: [
  {
    label: "GitHub",
    href: "https://github.com/refinex-lab/Refinex-Blog",
    icon: "github",
  },
],
```

站点还支持 iconfont 图标，用于导航页等模块。在 `iconfont` 字段配置 Symbol 模式的脚本地址即可：

```typescript
iconfont: {
  scriptUrl: "//at.alicdn.com/t/c/font_xxx.js",
},
```

## 个人信息（profile.ts）

`profile.ts` 控制「关于」页面的所有内容：

```typescript
export const profile: Profile = {
  name: "沉默的老李",
  title: "后端开发工程师",
  location: "中国 · 上海",
  avatarUrl,                    // 从 src/assets/profile/ 导入
  motto: "保持好奇，持续构建。",
  bio: "我专注于高并发服务、分布式系统与稳定性建设...",
  focus: ["分布式架构", "服务治理", "性能调优", "开发者体验"],
  skills: [/* ... */],
  projects: [/* ... */],
  experiences: [/* ... */],
};
```

头像图片放在 `src/assets/profile/` 目录下，通过 `import` 引入。

## 导航书签（navigate.ts）

`navigate.ts` 定义导航页的分类和站点链接：

```typescript
export const navigateConfig = {
  categories: [
    { id: "ai", label: "AI 助手", description: "对话、搜索与智能助理" },
    { id: "frontend", label: "前端文档", description: "框架、语言与工程化工具" },
    // ...
  ],
  sites: [
    {
      id: "chatgpt",
      title: "ChatGPT",
      description: "OpenAI 的对话式 AI 助手。",
      href: "https://chatgpt.com/",
      categoryId: "ai",
      pinned: true,
      iconSrc: "icon-OpenAiLogo",   // iconfont Symbol ID
    },
    // ...
  ],
};
```

图标支持三种方式：

- **iconfont Symbol**：设置 `iconSrc` 为 iconfont 图标 ID（如 `icon-github`）
- **本地图片**：将 SVG 放到 `public/navigate-icons/`，设置 `iconSrc` 为路径
- **外链图片**：设置 `iconSrc` 为完整 URL
