# 本站简介

这是一个基于 **Vite + React + TailwindCSS** 的静态文档/博客站点，目标是尽可能复刻 OpenAI Developers 文档的阅读体验：

- 左侧：文档目录树（支持折叠/展开、站内搜索）
- 中间：文档阅读区域（Markdown/MDX 渲染、代码高亮、Mermaid 图表、图片大图预览）
- 右侧：页面目录（On-page TOC，跟随滚动高亮）

## 为什么是“内容驱动”的方式

本站的文档主要来自 `content/` 目录下的文件树结构：文件夹即分类，`.md`/`.mdx` 文件即页面内容。

你只需要把笔记放进 `content/`，就能自动生成：

- 左侧目录树
- 搜索索引
- 访问路径（URL）
