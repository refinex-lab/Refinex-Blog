# 快速搭建

## 1. 克隆项目

```bash
git clone <your-repo-url>
cd Refinex-Blog
```

## 2. 安装依赖

```bash
npm i
```

## 3. 本地开发

```bash
npm run dev
```

启动后访问终端输出的本地地址即可。

## 4. 生产构建与本地预览

```bash
npm run build
npm run preview
```

`build` 会输出到 `dist/`，`preview` 会启动一个本地静态服务器用于检查最终产物。
