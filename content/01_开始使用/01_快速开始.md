# 快速开始

本项目的文档内容约定放在 `content/` 目录下（支持中文目录与文件名）。

## 1. 新建一篇文档

在 `content/Java/基础入门/` 里新增一个 Markdown 文件，例如：

```
content/
  Java/
    基础入门/
      基本介绍.md
```

## 2. 写点 Markdown

支持 GFM：表格、任务列表、删除线等。

- [x] 目录树自动生成
- [x] 搜索（支持中文）
- [ ] 未来可扩展：MDX、代码块增强、图表等

## 3. 代码块与复制

```ts
import { createRoot } from "react-dom/client";

createRoot(document.getElementById("root")!).render(<App />);
```

## 4. 下一步

- 给文档加上 `---` frontmatter，可以控制标题与排序。
- 你也可以直接用一级标题 `# ...` 作为标题。
